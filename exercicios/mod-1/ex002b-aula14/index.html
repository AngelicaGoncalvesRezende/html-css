Parágrafos e suas quebras

Você não consegue trocar de linha dentro do mesmo parágrafo apenas dando "enter" ao escrever

Você pode trocar de linha abrindo uma nova tag P ou usando a tag BR
para isso.

OBS.: Nunca usar várias tags BR para ter um garnde espaço entre os textos. Use o CSS para modificar esses detalhes da forma correta.

Agora, adicionando alguns símbolos especiais:

® © ™ € £ ¥ ¢ Δ ↑

Também é possível adicionar emoji:
Entre no site emojipedia, encontre o emoji desejado e encontre o seu código para incluir na sua página.
Sempre adicione antes do código os caracteres "&#x"!
