Principais formatações

Negrito e Itálico

Para enfatizar uma palavra é necessário usar a tag EN, fazendo com que a palavra ou frase fique enfatizada. Deve ficar assim!

Texto Marcado

Também é possível fazer com que um texto fique marcado com a tag MARK. Só é possível mudar a cor com CSS. Deve ser usada assim

Texto grande e pequeno

É possível deixar as letras grandes com a tag BIG, mas já está obsoleta e não é recomendado usar. Portanto, só é oissível fazer através do CSS ou usando algum nível de título (H1 - H6)

Para deixar a letra menor, há a tag SMALL. A qual é possível usar, pois não está obsoleta. Deve se usar assim

Texto inserido
